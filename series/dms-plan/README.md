# DMS Plan

Repository and series-about page for the `DMS Plan` series.

## What This Series Is

`DMS Plan` is an operations and workflow series for data management, workflow design, and repository delivery.

It is part of Steve Slevinski's professional research-support work on the current bridge site. It is not a SignWriting theory lane.

This series is for:

- research teams
- labs
- departments
- investigators preparing or revising data management and sharing plans
- teams moving from project systems to repository-ready delivery
