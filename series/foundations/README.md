# Sutton SignWriting Foundations

Repository and series-about page for the `Sutton SignWriting Foundations` series.

## What This Series Is

`Sutton SignWriting Foundations` is the core conceptual series for the Sutton SignWriting platform.

It gives the main public argument in three readable entries:

- language is not reducible to sound
- literacy is a durable symbolic layer beyond primary language access
- writing is not inherently linear
- signed-language writing opens a different conceptual and practical horizon

This series is intentionally public-facing. Evidence limits, research design, technical specification, and literature review belong in the specialist series, not as apparatus attached to these essays.

## Public Reading Path

Readers should begin with:

- `Sutton SignWriting Foundations: Start Here`

Then read:

- `Sutton SignWriting Foundations: Language Without Sound`
- `Sutton SignWriting Foundations: The Shape of Writing`

`Start Here` now carries the plain-language orientation directly. The two anchor essays then develop the conceptual argument.

## Series Contents

- `entries/start-here/` - plain-language orientation to the series and the core public argument
- `entries/language-without-sound/` - public conceptual paper on language, modality, literacy, and durable pathways
- `entries/the-shape-of-writing/` - public conceptual paper on writing, space, and plane-based text
