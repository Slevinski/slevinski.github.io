# Sutton SignWriting Infrastructure and Program

Repository and series-about page for the `Sutton SignWriting Infrastructure and Program` series.

## What This Series Is

`Sutton SignWriting Infrastructure and Program` is the support, continuity, and stewardship lane of the Sutton SignWriting platform.

It explains what already exists, what remains fragile, and what kind of technical, human, institutional, and funding support would make the work durable and cumulative.

It now carries the support framing that used to be scattered across older combined materials.

The clearer support-facing question is:

> what structure lets a real international language infrastructure survive, improve, and support research?

This series is for:

- PIs
- university units
- research infrastructure collaborators
- funders
- stewards and long-term supporters
