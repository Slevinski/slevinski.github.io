# Notebook

Repository and series-about page for the `Notebook` series.

## What This Series Is

`Notebook` is a low-prominence working-notes series.

It preserves material that is valuable, generative, and not yet settled enough for the strongest public platform series.

It is not a dumping ground and should not become the main public route into the platform.

## Public Reading Path

Public readers should begin with:

- `Notebook: Notebook Overview`
- `Notebook: Opening Note`

The full public overview now lives as a series entry:

- [entries/overview/essay.md](./entries/overview/essay.md)

## Series Contents

- `entries/overview/` - public overview for the working-notes area
- `entries/opening-note/` - opening note for the purpose and boundaries of the notebook

## Publication And Citation Model

The likely publication model is:

- overview: web-first series overview
- notebook entries: web-first working notes
- mature pieces: graduate into a stronger public series before archival publication

Notebook material should become DOI-bearing only after it has been promoted into a stable artifact with a clear scholarly role.
