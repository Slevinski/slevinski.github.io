# Comparative Matrix: Signed Language Writing Evaluations

This matrix is the compact companion to *A Layer-Based Evaluation of Signed Language Writing Systems*.

It is intentionally bounded to the same **core four-case demonstration set**:

- Filhol (2020)
- Grushkin (2017)
- Thiessen (2011)
- Unicode SignWriting

It does not replace the fuller series, and it does not attempt to summarize every review now present in the project.

Its purpose is narrower:

- to make the recurring comparative pattern visible quickly
- to show how the framework behaves across four different problem locations

Later entries such as Martin (2000) and Meletis and Durscheid (2022) extend the series beyond this core demonstration set.

## Key pattern

Across all four cases, the best unifying statement is not that they all make exactly the same mistake.

The stronger statement is:

> the writing problem is repeatedly displaced

In each case, one real layer becomes a stand-in for the whole:

- semantics and software representation
- literacy transfer and alphabetic compatibility
- symbol structure
- character standardization

Where a writing-system or text model answer is actually required, the field repeatedly fails to stabilize:

- the written sign as the primary written unit
- spatial composition as part of authored text
- and a clean separation of writing, encoding, and rendering
