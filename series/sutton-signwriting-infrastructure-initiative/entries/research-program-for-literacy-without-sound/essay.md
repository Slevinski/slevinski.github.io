# Research Program for Literacy Without Sound

**What could be studied next, why Brazil matters, and what support would make it possible**

The science framing already does important work.

It establishes that language is not tied to sound, that literacy is a durable symbolic expansion rather than a replacement for language, and that signed writing deserves serious scientific attention.

But if the argument stops there, it risks remaining permanently conceptual.

The stronger next move is a concrete SignWriting-centered research program.

This does **not** need to become a script-comparison contest.

It also does **not** need to begin with large and expensive neuroscience promises.

The more useful question is:

> what practical research program can begin now, with real populations, real materials, and a real data-and-support plan?

## Why a concrete program is worth doing

A more concrete program would help in several ways.

- It would move the conversation from possibility to evidence.
- It would make the support case easier for institutions and funders to understand.
- It would connect the science series to a buildable research agenda.
- It would let Brazil function as a real empirical base rather than only as background evidence.
- It would produce reusable materials, datasets, and study designs that later work can build on.

That last point matters most.

The best scientific program is not only one that produces findings.

It is one that produces **reusable capacity**.

## What this program is not

The most useful boundaries are as important as the ambitions.

This program is not:

- a generic comparison of all sign-language scripts
- a project organized around defeating ASL-phabet or any other system
- an attempt to settle every literacy question immediately
- a promise that neuroscience will answer everything first

The better lane is narrower and more constructive:

- build evidence for SignWriting as readable and writable text
- study how durable signed text is used
- connect those studies to real educational and institutional contexts
- build the infrastructure that lets the science accumulate

## Three research lanes

The cleanest structure is a three-lane program.

### 1. Reading and literacy lane

This lane asks what happens when signed language gains durable readable and writable text.

The central questions include:

- how SignWriting readers experience comfort and comprehension
- how written sign supports annotation, revisiting, and revision
- how text use differs from relying on video alone
- what kinds of written materials are easiest or hardest to use

This lane stays close to the core claim of *Language Without Sound*:

literacy changes what language users can do.

### 2. Brazil as empirical center

This lane uses Brazil as the strongest current real-world base.

Brazil matters because it already offers:

- documented institutional density
- repeated teaching practice
- written materials
- a deeper public research record than many English-language readers realize

That makes Brazil the natural place to begin:

- institutional mapping
- partnership building
- materials collection
- early reading and literacy studies

### 3. Infrastructure and dataset lane

This lane makes the science cumulative.

It asks:

- what texts should be collected
- what corpora should be built
- what reading stimuli and tasks can be standardized
- what annotation and search tools are needed
- what documentation is required so the work can be reused

Without this lane, every study risks becoming isolated.

With it, the field starts to build on itself.

## Realistic early studies

The first phase can be practical and still scientifically valuable.

### Institutional mapping

Update the Brazil accounting.

Identify active institutions, teachers, written materials, learner populations, and current practices.

### Materials audit

Collect the written artifacts already in use:

- classroom materials
- books
- short texts
- dictionaries
- worksheets
- online course materials

### Reader-experience study

Study how Deaf SignWriting readers describe:

- reading comfort
- comprehension
- navigation
- annotation
- cultural identification

### Short text and task pilot

Design small studies around:

- isolated signs
- short passages
- instructional texts
- narrative texts

The goal is not to prove everything at once.

The goal is to learn what kinds of written stimuli are most useful and measurable.

### Corpus-building pilot

Assemble a reusable written-Libras dataset that later literacy and cognition work can draw from.

That could include:

- normalized text
- metadata
- source context
- basic annotation

## Who and what should be studied first

The strongest early populations are already visible if the program stays focused.

### Populations

- Deaf signers with early sign-language access
- current SignWriting learners
- SignWriting teachers and course designers
- Deaf academic readers where suitable

### Materials

- SignWriting classroom texts
- books and literature
- dictionaries and lexical materials
- short curricular and instructional passages

That combination is realistic.

It also keeps the work anchored in people who are already using or teaching SignWriting rather than waiting for a hypothetical future population.

## Why Brazil matters inside this program

Brazil is not included here as a symbolic success story.

It matters because it already makes the program more feasible.

The Brazilian record:

- reduces dependence on purely hypothetical users
- offers a stronger starting point for partnerships
- gives access to written materials and teaching practice
- makes longitudinal and cross-site studies more imaginable
- supports a stronger grant narrative

This is why Brazil belongs inside the program design rather than only in background justification.

## What success looks like in phase one

The first phase succeeds if it produces durable research assets.

That means:

- an updated institutional map
- a curated materials set
- a reusable pilot corpus
- early reader-experience findings
- better-defined later-stage literacy and cognition questions

The point is not to end the debate.

The point is to make later work stronger, cheaper, and more cumulative.

## Why this belongs in the infrastructure initiative

This document belongs here because it is a bridge between the science argument and the practical work needed to study it.

It translates the scientific framing into:

- study design
- partnership needs
- data needs
- support needs
- institutional value

That is exactly what the infrastructure series is supposed to do.

## Bottom line

A more concrete scientific program is a good idea if it stays disciplined.

It should:

- remain centered on SignWriting rather than comparison contests
- use Brazil as the strongest documented current empirical base
- begin with realistic studies and reusable materials
- let infrastructure make the science cumulative

That is specific enough to start now and strong enough to justify real institutional support.
