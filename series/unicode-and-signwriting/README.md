# Unicode and SignWriting

Repository and series-about page for the `Unicode and SignWriting` series.

## What This Series Is

`Unicode and SignWriting` is a focused reference and briefing series for Unicode history, compatibility, implementation, Wikimedia context, and standards-facing SignWriting discussion.

It is not the main public entry point into SignWriting.

It exists because Unicode questions around SignWriting keep returning and need a stable document trail.

This series is for:

- developers
- standards readers
- Unicode and Wikimedia readers
- platform maintainers
- people evaluating compatibility between official Unicode SignWriting and the Sutton SignWriting production ecosystem
